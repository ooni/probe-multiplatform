网站和社交媒体应用被屏蔽了吗？您的网络是不是慢得不同寻常？运行 OONI 来测试一下吧。

借助这个应用，您可以检查网站和即时通讯应用的屏蔽情况，测量您的网速和性能，检查您的网络中是否存在负责审查和监视的系统。

OONI Probe 由 Open Observatory of Network Interference (OONI) 开发，是隶属于 Tor Project 的一个自由软件项目，旨在揭露全球网络审查状况。

OONI 全球社区自 2012 年起在 200 多个国家/地区收集了数以百万计的网络测量数据，揭露了多起网络审查案例。

收集网络审查的证据
您可以检测网站和即时消息应用屏蔽与否以及其屏蔽方式。您收集的网络测量数据可能会作为网络审查的证据。

检测用于审查与监控的系统
OONI Probe 测试还被设计用来发现可能用于审查和监控的系统（middlebox）。

测量您的网速与性能
您可以通过运行 OONI 的网络诊断测试（NDT）来测量您的网速和性能。您还可以使用基于 HTTP 的动态自适应流媒体（DASH）测试来测量视频流性能。

开放数据
OONI 发布网络测量数据，是因为开放数据允许第三方验证 OONI 发现、开展独立研究和解答其它研究问题。公开发不的 OONI 数据还可以帮助增加全球网络审查的透明度。您可以在这里探索和下载 OONI 数据：https://ooni.io/data/

自由软件
所有 OONI Probe 网络诊断（包括 NDT 和 DASH 实现）都基于自由和开源软件。您可以在 GitHub 上找到 OONI 软件项目：https://github.com/ooni 。想知道 OONI Probe 测试是如何工作的吗？详情请看：https://ooni.io/nettest/
