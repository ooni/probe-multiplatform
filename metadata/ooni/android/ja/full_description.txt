ウェブサイトや SNS アプリがブロックされていますか？ネットワークが異常に遅いですか？OONI Probe を実行して調べましょう！

このアプリを使えば、ウェブサイトやメッセージアプリのブロック状況を調査したり、ネットワークの速度やパフォーマンスを計測したり、検閲や監視に関与している可能性のあるシステムがネットワーク内にあるかどうかを確認したりすることができます。

OONI Probe は、世界中のインターネット検閲を調査することを目的としたフリーソフトウェアプロジェクト (Tor Project 傘下) である Open Observatory of Network Interference (OONI) によって開発されました。

2012 年以来、OONI のグローバルコミュニティは 200 カ国以上から数百万件のネットワーク測定データを収集し、数多くのネットワーク干渉の事例を明らかにしてきました。

インターネット検閲の証拠を収集する
ウェブサイトやメッセージアプリがブロックされているかどうか、またどのようにブロックされているかを確認できます。収集されたネットワーク測定データは、インターネット検閲の証拠となる可能性があります。

検閲や監視に関与しているシステムを検出する
OONI Probe のテストは、検閲や監視に関与している可能性のあるシステム (Middlebox) の存在を検出するようにも設計されています。

ネットワークの速度とパフォーマンスを計測する
OONI が実装した Network Diagnostic Test (NDT) を実行することで、ネットワークの速度とパフォーマンスを計測できます。また、Dynamic Adaptive Streaming over HTTP (DASH) テストを使用して、動画ストリーミングのパフォーマンスを測定することも可能です。

オープンデータ
OONI はネットワーク計測データを公開しています。これにより、第三者が OONI の調査結果を検証したり、独自の研究を行ったり、その他の研究課題に取り組んだりすることが可能になるからです。また、OONI のデータを公開することは、世界中のインターネット検閲を明るみに出すことにもつながります。OONI のデータの閲覧およびダウンロードはこちら: https://ooni.io/data/

フリーソフトウェア
OONI Probe のすべてのテスト (NDT および DASH の実装を含む) は、フリー・オープンソースソフトウェアに基づいています。OONI のソフトウェアプロジェクトは GitHub でご覧いただけます: https://github.com/ooni OONI Probe のテストの仕組みに興味がありますか？詳細情報: https://ooni.io/nettest/

OONI からの最新情報を受け取るには、Twitter でフォローしてください: https://twitter.com/OpenObservatory
